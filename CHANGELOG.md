# Changelog

All notable changes to Dynamic Source Loader will be documented in this file.

## [1.0.0] - 2025-12-22
### Added
- Initial release with dynamic source loading
- Detachable renderer panel
- Auto-reload functionality
- Background validation
- Professional dark theme UI
- Configuration file support (.ini)
- Error handling and reporting
- Memory usage monitoring
- State persistence

### Features
- Dynamic loading of Python-Qt applications
- Safe preflight validation
- File system watching for auto-reload
- Dependency parsing and checking
- Scoped styling (host vs loaded apps)
- Menu and toolbar controls
- Status bar with real-time info
- Settings persistence
